# Introdução

Uma rede fictícia de lojas precisa de um dashboard de performance de vendas para a diretoria.


*Tecnologias utilizadas incluen Power Query para limpeza e análise de dados e PowerBI para criação de dashboards.*

*data-set utilizado:*
[Baixar Excel](/static/images/vendas_techwave.csv)

# Demanda para Análise de Dados
Guilherme, como analista de dados, preciso que você responda:

1. **Visão Geral de Vendas**

Qual o faturamento total no período?

Qual a quantidade de produtos vendidos?

Qual o ticket médio por venda?
 
2. **Análise por Produto**

Quais são os top 10 produtos mais vendidos (em quantidade e em faturamento)?
 
3. **Análise por Loja**

Qual loja fatura mais?

Qual loja tem maior ticket médio por cliente?
 
4. **Análise de Clientes**

Quantos clientes únicos compraram no período?

Qual a idade média dos clientes por loja?

Existe diferença de consumo entre faixa etária (ex.: jovens x adultos x idosos)?

Quem são os clientes VIP (top 5% que mais gastaram)?

5. **Tendência Temporal**

Evolução de vendas por mês/semana/dia.

Existe sazonalidade (picos de vendas em determinados meses)?

# Metodologia
1. **Coleta de Dados**: Importação dos dados de vendas, produtos, clientes e lojas.
2. **Limpeza de Dados**: Tratamento de valores nulos, formatação de datas e remoção de duplicatas.
3. **Análise Exploratória**: Uso de tabelas dinâmicas e gráficos para identificar padrões.
4. **Cálculo de Métricas**: Cálculo de faturamento, ticket médio, lucro bruto, etc.
5. **Visualização**: Criação de dashboards interativos no PowerBI para facilitar a interpretação dos dados.
