# Introdução

A FitLife Academias tem filiais em várias cidades do Brasil. Nosso objetivo é entender melhor o comportamento dos alunos
para reduzir a evasão (cancelamento de planos) e aumentar a retenção. 

*Tecnologias utilizadas incluen Excel para limpeza, análise de dados e criação de dashboards, Power BI para visualização
 de dados e criação de relatórios interativos.*

*data-set utilizado:*
[Baixar Excel](/static/images/alunos_fitlife.csv)

# Demanda para Análise de Dados
Guilherme, precisamos que você analise os dados de nossos alunos para responder:
1. **Perfil dos alunos mais fiéis** → Quais características (idade, gênero, cidade, plano contratado, frequência de treinos)
 estão mais relacionadas com alunos que permanecem por mais tempo?
2. **Fatores da evasão** → Quais padrões você encontra nos alunos que cancelaram?
3. **Uso das academias** → Qual é a média de frequência dos alunos por cidade? Alguma cidade está com desempenho abaixo do esperado?
4. **Sugestões estratégicas** → Baseado nos dados, o que você recomenda para aumentar retenção?

**data-set:** Refere-se ao conjunto de dados disponibilizado para análise, contendo informações sobre os alunos da FitLife Academias

# Metodologia
1. **Coleta de Dados** → Obtenção dos dados dos alunos, incluindo informações demográficas, histórico de planos, frequência de treinos e status de cancelamento.
2. **Limpeza de Dados** → Tratamento de dados ausentes, inconsistências e formatação para garantir a qualidade dos dados.
3. **Análise Exploratória** → Utilização de técnicas estatísticas e visualizações para identificar padrões e tendências nos dados.
4. **Analise de Retenção e Evasão** → Comparação entre alunos que permaneceram e os que cancelaram para identificar fatores-chave.
5. **Segmentação de Alunos** → Agrupamento dos alunos com base em características similares para entender diferentes perfis.
6. **Recomendações** → Desenvolvimento de estratégias baseadas nos insights obtidos para melhorar a retenção e reduzir a evasão.

# Resultados
## [✅] Perfil dos alunos mais fiéis:
![Github Traffic](/static/images/desempenho_por_rede.png)
Dentre as características existentes dos alunos, o tipo de contrato do aluno com a academia foi o fator que mais destacou os 
 fiéis dos demais. No gráfico acima, nota-se que o plano Anual se destaca com o maior número de meses Ativos.

## [✅] Fatores de evasão:
![Github Traffic](/static/images/desempenho_por_zona.png)
Além do tipo de Plano ressaltar os alunos que são mais e menos ativos nas academias, não teve outra características que se destacou 
para tirar insights sobre evasão (gênero, idade, frequência). No entanto, teve algo curioso, os alunos com grande engajamento 
(frequentavam a academia entre 11 - 20 dias no mês) são também os que mais tem a matrícula cancelada.

## [✅] Uso das academias:
![Github Traffic](/static/images/desempenho_por_numero_de_alunos.png)
São Paulo é a cidade que se destaca de forma negativa, pois é o local com que os alunos menos frequentam a academia mensalmente. 
Rio de Janeiro se destaca por ter os alunos mais engajados. Sobre São Paulo, deve-se ligar o sinal de alerta (❗), pois baixa 
frequência dos alunos significa que os matriculados não estão empenhados e motivados o suficiente para frequentar a academia e 
isso pode ocasionar em cancelamento de matrículas. 

## [✅] Sugestões Estratégicas:
![Github Traffic](/static/images/desempenho_por_rede_plus.png)
1. **Visto que clientes com plano anual tende a ter mais meses ativos nas academias, é interessante tentar incentivar novas alunos a 
optar pelos planos anuais**, a partir de um bom incentivo para isso (brinde, desconto imperdível, preferências…). Além disso, não só ter
 estratégia com novos alunos, mas também tentar fazer com que os alunos ativos com outro tipo de plano, opte pelo plano anual, haja 
 visto as vantagens do plano anual.
2. No gráfico abaixo, note que os alunos com matrícula mensal são os que mais cancelam e também são os que tem mais em quantidade de 
alunos. Nesse sentido, **transferir esses alunos mensais para o plano trimestral ou, ainda melhor, anual, já seria uma boa estratégia 
para diminuir a evasão**.
3. O caso dos alunos engajados que cancelam suas matrículas devem ser investigados.
**Vão para um concorrente?**
**Os equipamentos não estão entregando o que devem para esses alunos?**
**Falta de direcionamento?**
**Valor da mensalidade acima do que cabe no seu bolso?**
**Atendimento disfuncional ou ineficiente?**
Deve-se, no ato do aluno cancelar a matrícula, fazer um questionário breve e anônimo do por que esses alunos estão cancelando. A partir
 dessas informações, teremos mais insumos para entender e iniciar ações para combater os motivos que incentivam os alunos ao 
 cancelamento de matrícula.
